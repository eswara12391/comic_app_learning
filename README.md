Quick start

1. Create a virtual environment:

```bash

```

2. Activate the environment and install requirements:

On Windows (PowerShell):
```powershell
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

3. Run the app:

```bash
python app.py
```
